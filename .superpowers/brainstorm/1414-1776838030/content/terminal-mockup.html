Comandos disponibles

  • help — lista de comandos
  • whoami — nombre, rol, ubicacion
  • ls projects — lista proyectos
  • skills — tecnologias dominadas
  • github --stats — datos en vivo via API
  • contact — email y redes
  • clear — limpia la terminal

Detalles tecnicos

  • 100% vanilla JS, sin dependencias
  • GitHub REST API (sin auth, limite 60 req/h)
  • Typing effect en cada respuesta
  • Historial con flecha arriba
  • Cursor global estilo dot sobre el fondo de estrellas
  • Tab-completion opcional para comandos

El widget se integra en index.html reemplazando los botones CTA del hero. El input del teclado va directo al campo oculto de la terminal.
